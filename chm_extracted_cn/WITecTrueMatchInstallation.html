安装


WITec TrueMatch 软件通过 WITec Project 的组合安装程序提供（或作为单独的 .msi 安装程序提供）。

它包含以下软件组件：

  • WITec TrueMatch 可执行文件和所有相关 DLL
  • WITec ParticleScout 和 WITec TrueMatch 集成在同一应用程序中


程序启动


当从 WITec Control 或 WITec Project 将光谱导出到 TrueMatch 时，TrueMatch 会自动启动。

您也可以使用 Windows 开始菜单或桌面上的"WITec ParticleScout"快捷方式启动 TrueMatch。


许可证


需要特殊的 TrueMatch 许可证才能无限制使用 TrueMatch。

一旦您订购了 TrueMatch 许可证，它将包含在 WITec Project 许可证中，可在任意数量的计算机上使用。


计算机要求


  • 最低 4 GB RAM，推荐 8 GB
  • 1 GB 硬盘空间
  • Direct3D 兼容的显示适配器
  • Windows 10 64 位
  • .NET Runtime 4.8 或更新版本
