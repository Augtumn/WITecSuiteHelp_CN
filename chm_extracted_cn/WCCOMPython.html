Python

对于 Python，WITec Control 随附了即用型的包，其中包含不同测量模式和操作的类。以下描述安装过程。进一步说明如何在不使用此包的情况下使用 Python。


WITec Python 包

前提条件

  • 安装最新 Python版本（>= 3.10）


包安装

WITec Python 包可以使用 pip 安装：

pip install WITecSDK


如果计算机未连接互联网，安装离线包及其依赖项（将 6.X 和 1.2.X 替换为当前的 WITec Suite 和包版本）：

  • pip install "C:\ProgramData\WITec\WITec Suite 6.X\Common Files\WITec.SDK\Python\WITecSDK-1.2.X.tar.gz"
  • comtypes — 轻量级 Python COM 库 — 推荐，测试更充分
  • pywin32 — 在 Python 下提供对许多 Windows API 的访问 — 备用方案（如未安装 comtypes）


示例

使用 WITec Python 包的示例代码：

from asyncio import run

from WITecSDK import WITecSDKClass


WITec = WITecSDKClass()

WITec.RequestWriteAccess()

videoImageAcquisition = WITec.CreateVideoControl()

bitmappath = run(videoImageAcquisition.AcquireVideoImageToFile())


更多示例可以在这里找到（将 6.X 替换为当前 WITec Suite 版本）：

C:\ProgramData\WITec\WITec Suite 6.X\Common Files\WITec.SDK\Python\Examples

  • LASExample.py：展示如何定义和启动不同类型的大面积扫描
  • LAScanfromFile.py：可用于在 WITec Control 中定义大面积扫描并使用WriteLAtoFile(filename)将设置保存到文件。该文件可用于通过RunLAfromFile(filename)自动运行预定义的扫描。此示例展示如何动态使用写访问。在测量时，可以关闭允许写访问到 WITec Control 的窗口，以便使用滤波器查看器等。程序将等待，直到再次按下控制树中的COM 自动化按钮以允许外部软件的写访问后，才启动下个大面积扫描。
  • ManualTopography.py：展示一种通过使用光谱自动对焦来自动化手动形貌校正的可能性。
  • HeatWITecTimed.py：展示WITec 加热台的使用以及如何将附加数据通道提供给慢速时间序列。此外，它为慢速时间序列实现了光谱自动对焦。
  • HeatWITecManual.py：与上一示例功能相同，但使用手动慢速时间序列。这允许单独触发下一次测量。
  • AdditionalParameter.py：展示如何将所有可用参数读取到文件以及如何访问特定参数。

不使用 WITec Python 包

前提条件：

  • 安装comtypes（pip install comtypes）

或

  • 安装pywin32（pip install pywin32）

使用 comtypes 的示例代码：

from socket import gethostname

from comtypes.client import GetModule, CreateObject

bucstlb_id = '{E84FEF4C-DD2F-4B8D-9C2F-D016FEBB01F4}'

GetModule((bucstlb_id, 1, 0))

from comtypes.gen.BasicUniversalCOMServerLib import *

# 连接到 WITec Control

wcCoreInterface = CreateObject(CBUCSCore, interface = IBUCSCore, machine = gethostname())

# 获取参数修改器

parameterModifier = wcCoreInterface.GetSubSystemDefaultInterface("UserParameters|SequencerAsTimeGoesBy|IntegrationTime").QueryInterface(IBUCSFloat)

# 读取浮点参数的值

value = parameterModifier.GetValue()

print(value)

# 请求写访问

# 前提条件：WITec Control 必须允许远程写访问（控制窗口，参数：COM 自动化 -> 允许远程访问）

wcCoreInterface.QueryInterface(IBUCSAccess).RequestWriteAccess(true)

# 写入浮点参数的值

parameterModifier.SetValue(0.42)


使用 pywin32 的示例代码：

from pythoncom import CLSCTX_REMOTE_SERVER, VT_ARRAY, VT_R4, VT_BSTR

from win32com.client import DispatchEx, CastTo, VARIANT

from socket import gethostname

CLSID = "{C45E77CE-3D66-489A-B5E2-159F443BD1AA}"

IBUCSAccess = DispatchEx(CLSID, machine = gethostname(), clsctx = CLSCTX_REMOTE_SERVER)

IBUCSCore = CastTo(IBUCSAccess, 'IBUCSCore')

parameterModifier = IBUCSCore.GetSubSystemDefaultInterface("UserParameters|SequencerAsTimeGoesBy|IntegrationTime")

parameterModifierFloat = CastTo(parameterModifier, 'IBUCSFloat')

value = parameterModifierFloat.GetValue()

print(value)

IBUCSAccess.RequestWriteAccess(True)

parameterModifierFloat.SetValue(0.42)

stringarray = VARIANT(VT_ARRAY | VT_BSTR, ["str1","str2"])

floatarray = VARIANT(VT_ARRAY | VT_R4, [5.2,8.4])
