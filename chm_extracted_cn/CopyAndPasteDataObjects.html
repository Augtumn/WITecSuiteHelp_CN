复制与粘贴数据对象

可以将选定的数据对象复制到 Windows 剪贴板中。您可以通过打开项目管理器的上下文菜单项"复制数据"（快捷键 Ctrl-C）来完成此操作。


请注意，启动任何 WITec Project 或 WITec Control 实例时，剪贴板将被清空。如果您打算将数据从一个 WITec Project 实例复制到另一个，必须在复制数据之前打开第二个实例。


只需使用项目管理器的上下文菜单项"粘贴数据"（快捷键 Ctrl-V）将复制的数据追加到项目。


如果您想复制和粘贴多个对象，请确保在一个步骤中粘贴它们；否则由于单个对象之间缺少关系，某些分析功能将无法工作。



粘贴位图

也可以使用"粘贴位图"功能将位图数据（如屏幕截图）复制到项目中。将创建一个不带变换的新位图对象并追加到项目。
