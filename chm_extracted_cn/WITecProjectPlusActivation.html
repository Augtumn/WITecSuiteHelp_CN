WITec Project Plus激活

请求报价


如果您对我们的高级数据评估软件WITec Project Plus感兴趣，请随时请求报价：info@WITec.de。



激活过程


在使用您的WITec Project Plus许可证之前，您需要先激活该许可证：


发送请求：

  • 在应激活Plus许可证的计算机上启动WITec软件。
  • 通过"主菜单 > 帮助 > 许可证管理器"打开许可证管理器。
  • 按下"请求WITec Project Plus密钥"按钮。
  • 输入您的联系信息。
  • 如果您购买了多个许可证以在多台计算机上激活软件，请在原因字段中输入计算机名称，以便以后区分密钥。
  • 如果同一台计算机上安装了电子邮件客户端，请按"发送请求电子邮件"按钮将请求发送给WITec；否则：
  • 按"保存请求到文件"按钮将请求存储在文本文件中，并将其复制到安装了电子邮件客户端的计算机上，然后将文本文件的内容复制到发送给key@witec.de的新电子邮件中。


接收并安装许可证：

  • WITec将通过电子邮件向您发送许可证密钥（密钥作为加密文本块发送）。
  • 在创建请求的计算机上打开WITec Project（或WITec Control）的许可证管理器。
  • 将加密文本块复制到密钥符号旁边的空白字段中，然后按"安装新许可证"。
  • 重新启动WITec Project。启动画面现在应显示WITec Project Plus已激活（"Plus版本"）。您可以随时通过打开WITec Project帮助菜单中的关于和支持面板来检查Plus版本是否激活。


请注意，WITec Project PLUS许可证仅适用于您创建许可证请求的计算机。


如果您在激活WITec Project Plus软件时遇到问题，请联系key@witec.de。
