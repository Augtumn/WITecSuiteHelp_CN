ST Japan Database

如何购买 ST日本 数据库


由于技术原因，无法直接从 ST日本 购买 ST日本数据库。

请直接联系 WITec 或当地代表，从 WITec 购买许可证和加密狗（价格无差异）。


ST日本数据库的安装


以下安装文件可供使用：


  1. ST日本 (全光谱), 版本 51
  2. ST日本 (全光谱), 版本 52 (较新版本，含更多光谱)
  3. ST日本 (全光谱), 版本 61 (较新版本，含更多光谱)
  4. ST日本 Psychoactive, 版本 52 (仅精神活性和药物物质)
  5. ST日本 Psychoactive, 版本 61 (仅精神活性和药物物质)


第1、2和3项的设置包含200个演示光谱的演示许可证。


注意，只有 WITec ST日本 安装文件才能与 TrueMatch / ParticleScout 配合使用，并且您必须安装与 ST日本 许可证匹配的版本。



ST日本许可证的安装


购买 ST日本 许可证后，您将获得一个 USB 硬件加密狗和一个名为"Access File"的文件。

Access File 的后缀为 ".stx"。

要安装 Access File，您可以打开 WITec TrueMatch，然后使用菜单 "TrueMatch -> 安装 ST日本 Access File"。浏览您的 Access File 并单击确定。

软件将询问您是否要安装 USB 加密狗驱动程序。如果之前未安装加密狗驱动程序，请单击是。


请确保在 "数据库 -> 管理数据库 .." 菜单中激活了 ST日本："显示 ST日本"。

现在 ST日本 许可证应可正常使用。

故障排除


如果 ST日本 无法正常工作，请考虑以下解决方案：


  • 确保 ST日本 USB 加密狗已正确插入。
  • 确保 USB 硬件插槽正常工作。
  • 检查 Windows 是否识别该 USB 加密狗。
    为此，请打开设备管理器并查找 "Software Security Token / USB Security Key"

    如果未列出该设备，请确保已安装 USB 加密狗驱动程序。
    这通常在使用 TrueMatch 菜单安装 Access 文件时自动完成。
  • 确保安装了正确版本的 WITec ST日本数据库 设置（51、52、...）
  • 确保安装了正确的 .stx Access File。
    在极少数情况下，如果您安装了多个 .stx 文件（例如，一个旧文件激活较少光谱，一个新文件激活更多光谱），您可能需要使用 TrueMatch 菜单中的 "浏览 ST日本 Access File 文件夹" 删除旧的/错误的 .stx 文件。
  • 确保 数据库 管理器中 "显示 ST日本" 复选框已勾选。
  • 检查 ST日本数据库 或任何可用类别是否已 被选中用于搜索。
