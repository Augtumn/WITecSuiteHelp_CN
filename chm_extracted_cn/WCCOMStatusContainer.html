状态容器

状态容器用于连接到 WITec Control。所有状态容器的路径以Status|开头，并实现为CBUCSStatusContainer。如果更新未实现的状态，应用程序返回 E_NOTIMPL。除非另有说明，数据方向是从服务器到客户端。


应用程序状态

路径        

Status|Software|Application|ProgramVersion

类型

单值，字符串

描述

应用程序的名称、版本以及（如果存在）构建版本。

路径        

Status|Software|Application|MemoryStatus|PhysicalMemory

Status|Software|Application|MemoryStatus|PageFile

Status|Software|Application|MemoryStatus|AddressSpace

类型

单值，整数

描述

相应值的可用空间百分比。

路径        

Status|Software|Application|CurrentFileName

类型

单值，字符串

描述

项目文件的名称（如果存在）。


序列器状态

路径        

Status|Software|Sequencers|IsASequencerActive

类型

单值，布尔

描述

如果序列器活动则返回 true。

路径        

Status|Software|Sequencers|ActiveSequencer|Name

类型

单值，字符串

描述

活动序列器的名称。如果没有序列器活动，返回 E_ACCESSDENIED。

路径        

Status|Software|Sequencers|ActiveSequencer|CurrentActivity

类型

单值，字符串

描述

序列器的当前活动（有关详细信息，请参考页面末尾的表格）。如果没有特定操作，返回 Sequence Busy。如果没有序列器活动，返回 E_ACCESSDENIED。

路径        

Status|Software|Sequencers|SequencerTimeSeriesSlow|UserDataCaptions

类型

一维数组，字符串

描述

外部用户数据通道的描述。

方向

客户端 -> 服务器

路径        

Status|Software|Sequencers|SequencerTimeSeriesSlow|UserDataUnits

类型

一维数组，字符串

描述

外部用户数据通道的单位。

方向

客户端 -> 服务器

路径        

Status|Software|Sequencers|SequencerTimeSeriesSlow|UserDataValues

类型

一维数组，浮点

描述

外部用户数据通道的值。

方向

客户端 -> 服务器

在启动慢速时间序列之前，所有三个参数（UserDataCaptions、UserDataUnits 和 UserDataValues）必须填充相同维度的数组。这些值在 GUI 中不可见。


数据通道状态

路径        

Status|Hardware|Controller|DataChannels|[数据通道名称]

类型

单值，浮点

描述

此数据通道的当前值（已考虑比例和偏移）。可用的数据通道取决于硬件配置，可以通过使用IBUCSCore的 GetSubSystemsList() 方法读取。


COM 子系统状态

路径        

Status|Software|COMAutomation|AcceptsRemoteConnection

类型

单值，布尔

描述

如果服务器允许写访问，则返回 true。

路径        

Software|COMAutomation|COMCallPerformance

类型

含两个值的一维数组，长整型

描述

索引 0：主机系统的 Tic 频率（低位）

索引 1：主机系统的 Tic 计数（低位）

用于测量两次 COM 调用之间的时间。


所有序列器的序列活动字符串

状态容器 Status|Software|Sequencers|ActiveSequencer|CurrentActivity 检索序列器的当前操作。除非另有说明，它可以传递以下当前活动字符串之一。（无法检索数值。）

序列活动字符串

数值

Sequence Busy

0

Sequence not Active

-1

Cleaning up after Sequence End

-2

Unknown Sequence Activity (n)

n

Moving Microscope to Focus-Position

3
