COM 自动化概述



COM 自动化功能支持 WITec Control 的远程控制。可以读取和写入所有参数，以及在控制窗口中按下按钮。视频窗口中的许多功能也可访问。


许可证要求：

  • COM 自动化

（如需为您的系统获取报价，请联系 WITec。）


什么是 COM？

组件对象模型（COM）是 Microsoft 开发的一种进程间通信技术。它可用于多种编程语言。


什么是 LabVIEW？

实验室虚拟仪器工程工作台（LabVIEW）是由National Instruments开发的图形化编程语言。它通常用于数据采集、仪器控制和自动化。



WITec 提供什么？

COM 自动化捆绑包包括：

  • 对各种编程语言可访问的 WITec Control COM 接口的读写权限
  • LabVIEW 驱动，包含一些文档化的示例
  • Python 和 C# 项目以便于使用


COM 接口可以通过各种编程语言访问，WITec 提供 LabVIEW 的文档化示例。对于 Python 和 C#，WITec Control 提供了即用型类的项目。关于 COM 接口结构的一般信息记录在接口部分。

允许对 WITec Control 的写入访问

默认情况下所有参数都可读取。要设置参数或触发操作，必须允许对 WITec Control 的写入访问。点击控制树的COM 自动化部分中的COM 自动化。这将打开一个窗口，阻止用户在 WITec Control 中进行输入，直到点击撤销。

可以通过启动 WITec Control 并运行随 WITec Control 安装的测试程序"WITec Control COM 连接测试程序"来检查 WITec Control COM 接口的正确功能。


请注意，WITec 当然会协助您解决由 WITec Control 或 WITec LabVIEW 驱动引起的错误或问题。但是，WITec 不会测试或纠正用户编写的代码。这些的正确功能完全由用户负责。
