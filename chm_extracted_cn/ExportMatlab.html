导出MATLAB

描述


MATLAB导出格式将任意类型的图谱数据对象导出为MATLAB结构体或DSO格式。

导出限制为2 GB数据。更大的数据可以使用 图谱ASCII或RAW导出。


输入和输出


输入：

一个或多个图谱数据对象（任意维度）。


输出：

一个或多个文件，每个文件包含每个选定数据对象的所有光谱



格式


根据MATLAB导出类型"DSO"或"Matlab结构体"，格式为以下之一。



DataSetObject（DSO 6.0）


此格式由"Eigenvector"指定。可以从那里下载MATLAB软件包。

参见 http://www.eigenvector.com/software/dataset.htm

它包含可用于DSO数据格式的MATLAB函数。它是免费的（BSD许可证）。

它仅支持用于数据显示的简单函数。

DSO作为MATLAB对象mat文件创建，参见http://www.mathworks.com/help/pdf_doc/matlab/matfile_format.pdf

每个DSO字段的说明可在以下地址获取：http://wiki.eigenvector.com/index.php?title=DataSet_Object

或在下载的软件包中。


DSO字段：


Name（名称）：

包含WITec Project项目管理器中图谱对象的名称。


Type（类型）：

图谱对象的类型。如果是2D图谱对象则为"image"，否则为"data"。


Date（日期）：

当前日期，格式为 yyyy.mm.dd hh.mm.ss


ModDate（修改日期）：

与Date相同。如果MATLAB更改了文件，将被修改。


ImageSize（图像大小）：

对于0D和1D图谱对象为Null，对于2D图谱对象为SizeY和SizeX


ImageMode（图像模式）：

如果是2D图谱对象则为1，否则为0


Data（数据）：

包含图谱数据。对于单个图谱对象，图谱数据保存为行向量。

对于0D和1D图谱对象（例如/时间序列/线扫描），图谱保存为连续的行向量。

数据存储在一个矩阵中，具有 <光谱数量> 行和 <光谱大小> 列。

对于2D图谱对象，矩阵具有 SizeY*SizeX 行和 <光谱大小> 列。

图像列顺序存储，然后存储行。


Label（标签）：

始终为null。


AxisScale（轴缩放）：

图谱缩放存储在具有2x2个元素的单元格中。为此使用导出过滤器选项中的X单位设置。

对于1D图谱（例如时间序列），单元格元素(1,1)包含序列缩放（例如时间缩放）。单元格元素(1,2)是序列的单位的字符串（例如"seconds"）。

对于单个光谱，单元格元素(1,1)和(1,2)为空。

单元格元素(2,1)始终是图谱x轴缩放，单元格元素(2,2)是x轴缩放的单位字符串（例如"rel. 1/cm"）。



ImageAxisScale（图像轴缩放）：

图像轴缩放存储在具有2x2个元素的单元格中。使用标准单位。

对于0D和1D图谱对象，元素为空。对于2D图谱对象，元素(1,1)包含图像列的缩放，元素(2,1)包含图像行的缩放，元素(1,2)和(2,2)包含x/y缩放的单位字符串（例如µm）。


Title（标题）：

始终为null。


Class（类）：

用于MATLAB识别文件为DSO格式。包含"dataset"。


Include（包含）：

用于设置图像或图谱掩模，告诉MATLAB应使用哪些像素。WITec Project始终在掩模中设置所有像素。

该字段是一个具有2个元素的单元格。元素1具有图像掩模，作为具有 <SizeY*SizeX> 个元素的行向量。

元素2具有图谱掩模，作为具有 <光谱大小> 个元素的行向量。


ClassLookUp（类查找）：

始终为null。


AxisType（轴类型）：

可用于表征光谱轴缩放（数据不会被此更改）。可以为"none"、"continuous"和"discrete"。WITec Project使用"none"。


ImageAxisType（图像轴类型）：

参见AxisType，但用于图像轴缩放。


Description（描述）：

始终为null。


UserData（用户数据）：

始终为null。


DataVersion（数据版本）：

WITec Project使用DSO 6.0，因此写入"6.0"。


History（历史）：

包含"Created by WITec Project (Version XXX)"。


UniqueID（唯一ID）：

为每个图谱对象从标题+日期生成唯一ID。



在MATLAB中使用DSO


下载的软件包包含一个 @dataset 文件夹。要在MATLAB中打开导出的DSO文件，文件必须与 @dataset 文件夹位于同一文件夹中（或者文件必须是此文件夹的同级文件）。

在MATLAB窗口"当前文件夹"中，可以双击文件或通过命令"load('filename_dso-file')"加载。

MATLAB随后在MATLAB窗口"工作区"中显示它。您可以通过输入DSO文件名在MATLAB命令窗口中列出DSO字段。

您可以使用字段，通过"dso-filename.fieldname"。具有多个值的字段（如AxisScale）可以通过例如"dso-filename.axisscale{2,1}"来访问。可以使用 @dataset 文件夹中的函数来处理DSO。

以下地址有相关文档：http://wiki.eigenvector.com/index.php?title=DataSet_Object_Methods



MATLAB结构体


这是由WITec定义的自定义结构体格式。

字段Name、Date、Data、AxisScale、ImageSize、ImageAxisScale与DSO格式相同。



在MATLAB中的使用


您可以在MATLAB的"当前文件夹"窗口中双击导出的文件或通过load命令来打开它。

结构体显示在MATLAB窗口"工作区"中。您可以双击结构体来显示字段，字段也可以被双击。

字段"date"、"data"和"imagesize"是矩阵，可以使用方括号访问，例如 structurename.data(1,:) 返回第一行数据，即数据集中的第一个光谱。

字段"axisscale"和"imageaxisscale"是单元格，通过花括号访问，例如 structurename.axisscale{2,1} 返回光谱轴缩放的行向量。

您也可以在输入"structurename."后按tab键来显示可能的字段。


以下是一些在MATLAB中绘制图谱对象的示例。只需将其中一个函数复制到MATLAB文件（filename.m）中并在MATLAB中加载即可：


function PlotGraph(aGraph, aImagePosition, aYLim)

% 在2D坐标系中绘制整个数据集中的一个图谱。

% aGraph: 来自WITec Project导出的结构体

% aImagePosition: 图谱的x和y图像坐标，例如 [30,50] 表示 x=30, y=50

% aYLim: 光谱y轴的下限和上限，例如 [-5,250] 将在 -5 到 250 的范围内绘制图谱

% 注意，由于自动缩放，瑞利峰可能会使您重要的拉曼峰不可见。


aSpectrum = aGraph.data(aImagePosition(1) * aGraph.imagesize(1) + aImagePosition(2),:);

aXAxisscale = aGraph.axisscale{2, 1};

plot(aXAxisscale, aSpectrum);

aXLabel = sprintf('[%s]',aGraph.axisscale{2, 2});

xlabel(aXLabel)

ylabel('CCD cts.')

ylim(aYLim)

aTitle = sprintf('%s (y = %i, x = %i)',aGraph.name, aImagePosition(2), aImagePosition(1));

title(aTitle)

end



function PlotAverageGraph(aGraph, aYLim)

% 绘制平均光谱

% aGraph: 来自WITec Project导出的结构体

% aYLim: y轴的下限和上限


aAverageSpectrum = mean(aGraph.data);

aXAxisscale = aGraph.axisscale{2, 1};

plot(aXAxisscale,aAverageSpectrum)

aXLabel = sprintf('[%s]',aGraph.axisscale{2, 2});

xlabel(aXLabel)

ylabel('CCD cts.')

ylim(aYLim)

aTitle = sprintf('%s (Average Spectrum)',aGraph.name);

title(aTitle)

end



function PlotImage(aGraph)

% 绘制强度图像

% aGraph: 来自WITec Project导出的结构体


aSizeY = aGraph.imagesize(1);

aSizeX = aGraph.imagesize(2);

aSumImage = sum(aGraph.data,2);

aImage = zeros(aSizeY,aSizeX);

for x=1:aSizeX

    for y=1:aSizeY

        aImage(y, x) = aSumImage((x - 1) * aSizeY + y);

    end

end

figure

aXAxisScale = aGraph.imageaxisscale{2, 1};

aYAxisScale = aGraph.imageaxisscale{1, 1};

imagesc(aXAxisScale, aYAxisScale, aImage)

%colormap(gray)

colormap(hot(256))

aTitle = sprintf('%s (Intensity image)',aGraph.name);

title(aTitle)

aXLabel = sprintf('[%s]',aGraph.imageaxisscale{2, 2});

xlabel(aXLabel)

aYLabel = sprintf('[%s]',aGraph.imageaxisscale{1, 2});

ylabel(aYLabel)

end



function SpectrumToImage(aGraph, aSpectralPosition)

% 绘制指定光谱位置的强度图像

% aGraph: 来自WITec Project导出的结构体

% aSpectralPosition: 光谱位置


aSizeY = aGraph.imagesize(1);

aSizeX = aGraph.imagesize(2);

aSumImage = sum(aGraph.data(:,aSpectralPosition),2);

aImage = zeros(aSizeY,aSizeX);

for x=1:aSizeX

    for y=1:aSizeY

        aImage(y, x) = aSumImage((x - 1) * aSizeY + y);

    end

end

figure

aXAxisScale = aGraph.imageaxisscale{2, 1};

aYAxisScale = aGraph.imageaxisscale{1, 1};

imagesc(aXAxisScale, aYAxisScale, aImage)

%colormap(gray)

colormap(hot(256))

aTitle = sprintf('%s (Intensity image)',aGraph.name);

title(aTitle)

aXLabel = sprintf('[%s]',aGraph.imageaxisscale{2, 2});

xlabel(aXLabel)

aYLabel = sprintf('[%s]',aGraph.imageaxisscale{1, 2});

ylabel(aYLabel)

end



导出过滤器选项


Matlab导出类型：

可以为DataSetObject (DSO 6.0) 或 Matlab结构体。


X单位：

导出的x轴数据的单位。可以为WITec Project能够处理的所有x单位（参见 解释数据）。
