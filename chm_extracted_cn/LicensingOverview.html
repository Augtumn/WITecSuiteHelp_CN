许可证概述

WITec Control 和 WITec Project 软件组件都需要有效的许可证密钥才能启动。


WITec Project

WITec Project 是分析 WITec 显微镜数据的基本后处理软件。WITec Project（基础版）许可证密钥随显微镜系统一起交付，可由该显微镜的整个用户组使用。

此密钥不绑定到特定计算机，可根据许可证产品在特定数量的计算机上安装。要将现有许可证密钥安装到新计算机上，请使用许可证管理器。


WITec Project Plus

WITec Project Plus 是 WITec Project 基础软件的附加组件，包含高级数据评估算法（参见WITec Project Plus 功能）。如果您没有许可证，所有 Plus 功能将以受限模式（演示模式）可用。

要不受限制地使用 Plus 功能，您需要购买绑定到计算机的许可证，参见WITec Project Plus 激活。此密钥严格绑定到一台计算机，不能安装在其他计算机上。


WITec Control

WITec Control 仅安装在由 WITec 提供的显微镜系统计算机上，应已安装有效的许可证。在（远程）软件升级的情况下，WITec 支持团队将安装此许可证。
