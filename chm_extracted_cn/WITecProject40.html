WITec Project 4.0
  • 外观和感觉
    • 新的启动/关于窗口
    • 桌面上和开始菜单中的新应用程序快捷方式图标，新的WIP文件图标
    • 新的快捷键查看器：显示软件的所有快捷键，参见菜单帮助 -> 显示快捷键
    • 新的拖放操作图标
    • 如果操作需要过多内存，用户现在会收到通知（某些创建大型图谱对象的拖放操作、项目管理器中的数据对象复制/粘贴）
    • 主窗口：
      • "关闭所有查看器和管理器"的新工具按钮，位于"重置查看器位置"按钮旁边
      • 最近的WIP文件直接在文件菜单中访问（无子菜单），菜单结构更优
    • 掩码图像对象的新图标（提取的绘制场图像或具有布尔单位名称的提取计算器图像）
    • 自动加载正确的窗口设置（适用于1个显示器）
  • 新的程序选项窗口（菜单选项 -> 程序选项）
    • 替换查看器定位选项、导出选项，并添加一些额外的新选项
    • OpenGL图形加速可以在新的程序选项中打开或关闭（对相同版本的WITec Project和WITec Control均有影响）
  • 内存扩展
    • 新的内存选项"内存映射文件"，允许程序处理超过2GB的数据对象（例如用于大型拉曼图像或堆栈；请参见菜单选项 -> 内存选项）
    • 建议使用64位操作系统、>=16GB内存以及快速SSD硬盘作为内存映射文件位置
  • 光标管理器：
    • 光标管理器不再是查看器，而是可见性可切换的工具窗口
    • 由于查看器中的新状态信息（见下文），光标管理器窗口默认不可见
    • 光标管理器窗口可以通过所有查看器中的快捷键"P"临时显示
  • 拖放操作窗口：
    • 拖放操作Plus窗口不再存在。所有Plus拖放操作都显示在标准拖放操作窗口中（如果Plus已激活）
    • 在项目管理器中启动拖放操作时，拖放操作窗口将自动显示在鼠标位置附近
    • 包含区域按钮不再存在。每个支持掩码的对话框现在都显示一个图像查看器。可以在此处绘制掩码，或分配（拖放）之前定义的掩码
  • 查看器通用
    • 自动鼠标模式
      • 带掩码的图谱查看器自动具有标记区域鼠标模式
      • 如果监听打开，图谱和图像查看器自动选择正确的鼠标模式，并在监听关闭时切换回移动模式。
    • 查看器定位更改
      • 如果某种查看器（例如图像查看器）的最后一个查看器被关闭，该查看器类型的定位将被重置
      • 加载项目后，所有查看器自动重新定位。在查看器定位选项中有一个打开/关闭此功能的选项
      • 自动阵列：如果没有足够空间容纳新的图谱或图像查看器，大小会自动调整
    • 图谱/图像/视频控制窗口中的新状态栏
      • 状态栏现在显示第二行，带有更多信息（图谱查看器：X光标位置，图像查看器和视频控制：空间光标距离）
      • 如果光标正在监听，状态栏中的标签显示为红色。单击红色标签将关闭该光标的监听模式
      • 一些常用功能可以在新状态栏中找到（例如图像自动色标、图谱缩放功能）
    • 当有人监听查看器的某个轴时，十字准线现在显示为红色
    • 当前显示数据的提取现在在所有图谱/图像查看器中都有效，并且始终有效（也在拖放操作预览和例如硬件光谱查看器中）。快捷键"Control + E"
    • 文本查看器：文本对象可以在同一查看器中切换（通过箭头按钮或快捷键F5/F6）
  • 新的滤波查看器（将替换旧的滤波管理器，也在WITec Control中）
    • 滤波不再是项目数据对象
    • 滤波与查看器一起存储
    • 可以将一组滤波存储到文件中以便快速访问（可选择对当前用户或所有用户）
    • 滤波范围由位置和宽度定义
    • 滤波位置/宽度可以监听：移动鼠标模式将更改位置，标记区域鼠标模式将更改位置和宽度
    • 预览图谱窗口
      • 显示局部平均图谱（如光谱到图像）和高亮显示的滤波位置
      • 高亮显示的滤波位置可以直接用鼠标移动模式移动
      • 标记区域工具打开弹出窗口，可以在标记范围内创建或更改新的滤波
    • 单个图谱在状态栏中有单值预览
    • 也适用于线图谱和具有空间X变换（横截面）的图谱
    • 输入图谱可以更改（仅当它具有相同大小和X变换单位类型时）
      • 将另一个图谱拖放到标题栏
      • 使用状态栏中的上一个/下一个按钮
      • 在滤波结果图像预览窗口中按Page Up / Page Down
    • 可以通过选择滤波来切换滤波预览。滤波列表中的复选框允许同时显示超过1个滤波。如果没有打开预览，双击滤波将打开预览。
    • 通过拖放到提取按钮进行批处理计算（拖放到"全部"按钮将计算并提取列表中的所有滤波）
    • 仅在按下提取操作之一时创建新数据对象
  • 图谱查看器
    • 新的缩放模式
      • 自动Y轴缩放、级联和同步缩放默认打开（您可以在图谱查看器程序选项中更改默认值）
      • 改进的级联函数，新的级联距离参数
      • 同步缩放，用于缩放所有图谱对象而不使用相同的Y轴
    • 新功能：可以显示峰值标签（使用峰值查找算法）。快捷键"F"
    • 图谱掩码数据对象现在可以直接拖放到图谱查看器中的掩码上以设置掩码
    • 图谱掩码可以隐藏。快捷键"H"
    • 新的跟随数据模式：光标将跟随当前显示图谱的精确支撑点
    • 新的直方图绘制避免了间隙和锯齿绘制问题（仅当X变换具有等距点时）；现在支持级联
    • 新快捷键：如果只显示1个图谱，Page Up / Page Down可在项目中的图谱对象之间切换
    • 图谱绘制选项现在可以同步（例如，为同一查看器中的所有图谱对象设置宽度）
    • 默认图谱绘制宽度可以在新的图谱查看器程序选项中定义
  • 图像查看器
    • 图像查看器也显示位图。位图查看器不再需要
    • 图像查看器可以显示没有空间变换的位图和图像
    • 新功能：同步缩放。将所有图像查看器缩放到相同的空间边界框。在同步模式中缩小将缩小所有查看器。快捷键"S"
    • 修订的新显示模式：
      • 3个输入图像槽：彩色图像、3D-Z图像、亮度图像；所有都可以通过拖放分配
      • 3D模式（切换，快捷键"3"），亮度模式（切换）
      • 3D图像模式现在自动使用透视投影和侧视图
      • 自由相机模式（切换，在3D模式下自动使用）：按住鼠标中键（滚轮）移动鼠标可以倾斜和旋转图像，可以使用鼠标滚轮定义自由相机距离
    • 十字准线可以隐藏（例如用于漂亮的演示）
    • 绘制场/掩码可以隐藏。快捷键"H"
    • 绘制场：绘制点模式的笔宽可以更改
    • 图像可以动画（在图像查看器程序选项中更改动画速度）
    • 可以绘制等高线（杂项视觉圆形菜单）
    • 漂亮的网格边框将在3D模式下关闭边框（视觉圆形菜单）
    • 线校正可以在新的状态栏右下角更改
    • 自动色标（以及通过右键单击切换）可以在新的状态栏右下角找到
    • 色标：
      • 底部/顶部色标现在可以互换
      • 底部/顶部色标现在可以同步更改
      • 在新的图像查看器程序选项中，可以调整自动色标算法的行为
    • 比例尺现在具有透明背景
    • 现在可以一次将多个对象拖放到查看器上以显示位置
  • 图像和图谱工具
    • 工具窗口不再存在
    • 图像和图谱查看器中的新圆形菜单：
      • 按住鼠标右键查看圆形菜单，然后向正确方向移动鼠标以选择其中一个圆形菜单
    • 所有功能都可以在新的圆形菜单中找到，包括以前只能在普通上下文菜单中找到的功能
    • 直接从图像/图谱查看器圆形菜单启动拖放操作对话框
  • 项目管理器
    • 切换对象可见性和排序以及启动拖放操作数据分析的新圆形菜单
    • 仅显示图谱、图像/位图、文本对象（高级模式显示所有对象）
    • 新的多重重命名工具，集成了重命名掩码和查找替换功能
    • 当某个查看器被激活/选中时，项目管理器现在会高亮显示该查看器使用的所有数据对象。在查看器定位选项中有一个打开/关闭此功能的选项
    • 追加项目时不加载项目管理器。在查看器定位选项中有一个打开/关闭此功能的选项
  • 拖放操作对话框：
    • 新对话框"图像直方图和统计"：图像直方图对话框现在也显示图像统计和粗糙度参数
    • 新对话框"图像变换和叠加"（Plus功能）：允许更改图像的空间变换，创建叠加位图，支持高级变换
    • 新对话框"图谱修复"（Plus功能）：允许手动插值所选光谱像素或范围，适用于任何类型的图谱数据对象中的某些或所有光谱
    • 新对话框"图谱和图像数据拼接"（Plus功能）：允许将一叠图像或图谱图像拼接/合并为一个更大的新数据对象
    • 新对话框"图像相关性"（Plus功能）：根据图像查看器中可更改的掩码显示相关点
    • 新的聚类分析用户界面：只有一个窗口，简化操作
    • 修订的背景扣除对话框
      • 新的扣除模式"最小值"。扣除所有光谱像素的最小值（使用掩码）
      • 新的扣除模式"常数"。可以扣除一个常数值或一个常数光谱
      • 新的扣除模式"形状"（Plus功能）。在光谱上拟合一个圆滑形状，实现优秀的背景扣除，即使对于荧光偏移也是如此
      • 现在支持横截面图谱对象
    • 图像横截面对话框：简化对话框，开始/停止坐标现在可以互换
    • 图像颜色组合：
      • 重新设计，具有简单/高级模式
      • 新模式"允许颜色混合"
      • 可监听编辑、与图像查看器的色标同步、用于更改色标的滚轮
      • 改进的透明度平滑参数
    • 光谱到图像：改进的监听模式
    • 图谱基础分析：简化对话框
    • 高级拟合工具：简化对话框
    • 图像滤波对话框不再显示错误图像，但有一个新的复选框用于临时显示原始图像（图像滤波、图像FFT 2D、图像背景扣除）
    • 数据分箱对话框现在称为数据裁剪和缩减。简化对话框；现在也允许裁剪位图
      • 版本4.0.12.9中的新功能：通过拖放进行批处理模式
    • 图谱分离器现在允许使用当前混合作为新的源图谱；允许具有相同X变换对象且相等的图谱
    • 仅一个图像和/或图谱对象的计算器对话框
    • 图像滤波对话框：新的图像平滑算法："各向异性平滑"
  • 类似的拖放操作现在合并为一个对话框
    • 图像统计/粗糙度拖放操作已合并到新的"直方图和统计"对话框中（图像直方图、图像统计、图像粗糙度）
    • 图谱滤波对话框已合并为一个对话框（图谱平均、图谱中值、图谱Savitzky Golay、图谱CRR）
    • 图像滤波对话框已合并为一个对话框（图像中值、图像平均、Plus版本图像滤波如Sobel等）
    • 图像背景扣除对话框已合并为一个对话框（N阶线扣除、N阶曲面扣除）
    • 图像堆栈对话框已合并为一个对话框（图像自动对焦、图像扩展对焦）
    • 图像FFT滤波已合并为一个对话框（图像FFT幅度、图像滤波FFT线、图像FFT 2D）
    • 图像线插值和图像移除坏数据对话框已合并为一个对话框
  • 导入/导出
    • 新的位图导入：位图可以通过菜单文件 -> 导入 -> 位图导入，或者如果位图在剪贴板中，在项目管理器中粘贴（Ctrl-V）
    • Tescan TIF导入（用于SEM图像，需要特殊许可证）
    • 导出到Matlab
    • 文本对象的批处理导出（ASCII和RTF）
